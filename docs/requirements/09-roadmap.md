## Phase 1：MVP 验证 — 核心引擎与 CLI 验证

**目标**：不写 UI / Daemon，搭建 `argus-core` 库并通过 `argus-cli` 验证。

| 任务 | 说明 | 优先级 |
|------|------|--------|
| Workspace 初始化 | 创建 Cargo Workspace 结构 | P0 |
| 核心数据结构 | `FileNode`, `Snapshot` 定义 | P0 |
| 扫描引擎 | 基于 `ignore` 库实现可中断扫描；Phase 1 使用同步 `Walk`，多线程 `WalkParallel` 留作性能优化 | P0 |
| 扫描引擎 | 基于 `ignore` 库实现可中断扫描；结果纯内存 | P0 |
| CLI 命令 | `scan` | P0 |
| 单元测试 | 核心算法单元测试 | P1 |
| 验收测试 | 端到端手动验证全流程 | P1 |

**交付物**：
- `argus-core` 库
- `argus-cli` 二进制
- 测试快照文件
- CLI 验收测试脚本

---

## Phase 2：TUI 极客版 — 视觉与交互落地

**目标**：基于 `ratatui` 开发终端交互界面。

| 任务 | 说明 | 优先级 |
|------|------|--------|
| TUI 框架搭建 | `ratatui` + `crossterm` 基础布局 | P0 |
| 文件树浏览器 | 基于方向键的目录树导航，支持当前树根 + scan size overlay | P0 |
| 列展示 | 名称 / 当前体积 | P0 |
| 排序切换 | 按名称 / 体积排序 | P0 |
| 配置加载 | 读取 `config.toml` 中的键位和主题 | P0 |
| Vim 快捷键 | 实现自定义键位映射系统 | P0 |
| 安全删除交互 | 二次确认弹窗 + 废纸篓调用 | P0 |
| 交互体验优化 | 实时进度条、ESC 取消、帮助面板 | P1 |

**交付物**：
- `argus-tui` 二进制
- 配置文件系统（keybindings + theme）

---

## Phase 3：Daemon 自动化 — 无感知实时监控

**目标**：实现 `argusd` 守护进程，事件驱动更新。

| 任务 | 说明 | 优先级 |
|------|------|--------|
| 文件系统监听 | `notify` 库集成 (inotify/FSEvents) | P0 |
| 去抖引擎 | 事件缓冲 + 延迟写入 | P0 |
| 轻量数据库 | SQLite 或 sled 集成 | P0 |
| UDS 通信 | Unix Domain Socket IPC | P0 |
| 历史快照管理 | 定时归档 + 保留策略 | P0 |
| TUI 升级服务模式 | TUI 通过 UDS 连接 Daemon | P0 |
| 客户端无感切换 | Standalone ↔ Server 自动识别 | P1 |

**交付物**：
- `argusd` 守护进程二进制
- 支持 `-2h` 相对时间查询的完整 TUI

---

## Phase 4：AI 完全体 — 智能观察窗与治理

**目标**：接入大模型，上线 AI 智能诊断能力。

| 任务 | 说明 | 优先级 |
|------|------|--------|
| async-openai 集成 | 异步调用大模型 API | P0 |
| AI 观察窗 UI | TUI 右侧实时渲染面板 | P0 |
| 特征提取引擎 | 数字指纹组装 + Prompt 生成 | P0 |
| Token 统计器 | 预估 + 实际计量 + 仪表盘 | P0 |
| 安全护栏 | AI 风险判定 + 强交互确认 | P0 |
| 多模型支持 | 兼容 OpenAI / ollama / vLLM | P1 |
| 配置页 UI | TUI 内 AI 配置界面 | P1 |

**交付物**：
- 具备 AI 诊断能力的完全体 `argus` 终端工具
- Token 消耗仪表盘

---

## Phase 5：GUI 桌面版 — 跨平台与生态系统

**目标**：图形化界面，系统菜单栏集成。

| 任务 | 说明 | 优先级 |
|------|------|--------|
| GUI 框架选型 | Slint 或 Tauri v2 | P0 |
| 矩形树图 (Treemap) | 可视化磁盘占用分布 | P0 |
| macOS MenuBar 集成 | 菜单栏图标 + 弹窗提醒 | P0 |
| 历史趋势折线图 | 磁盘空间变化时间轴 | P1 |
| 系统通知 | 空间阈值触发气泡提醒 | P1 |
| 自动更新 | 版本检查与更新推送 | P2 |

**交付物**：
- 正式发布的桌面级独立 App（.dmg / .AppImage / .deb）
